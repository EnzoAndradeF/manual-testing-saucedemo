# **Requisitos funcionais para testes**

## Módulo: Autenticação do usuário
- RF01 - O sistema deve permitir que o usuário realize o login com nome de usuário e senha válidos
- RF02 -  O sistema deve exibir mensagem de erro em caso de login inválido

## Módulo: Catálogo de produtos
- RF01 - O cliente pode ordenar o catálogo por nome de (A a Z) e (Z a A)
- RF02 - O cliente pode ordenar o catálogo por preço de (Menor valor para Maior valor) e (Maior valor para Menor valor)
- RF03 - O produto no catálogo deve exibir imagem, título, descrição e preço referente ao produto
- RF04 - O produto no catálogo deve ter a opção de adicionar no carrinho de compras
- RF05 - Ao adicionar produto no carrinho, um contador deve ser exibido no ícone do carrinho

## Módulo: Carrinho de compras
- RF01 - O cliente tem a opção de remover o produto do carrinho
- RF02 - O cliente tem a opção de alterar a quantidade do produto no carrinho

## Módulo: Checkout: Suas informações
- RF01 - Os campos nomes e sobre nomes devem ser validos apenas caracteres do tipo letra e o campo caixa postal deve apenas caracteres do tipo número
- RF02 - Os campos devem ser preenchidos para prosseguir com a compra

## Módulo: Checkout: Visão geral
- RF01 - O sistema deve mostrar o valor total do meu pedido de compra

## Módulo: Confirmação
- RF01 - Uma mensagem de confirmação deve ser exibida após finalizar a compra
